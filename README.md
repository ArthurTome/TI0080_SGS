### Manual
1. Instalar pacotes
``` bash 
~/TI0080_SGS/app$ npm i
~/TI0080_SGS/app$ npm install nodemon
~/TI0080_SGS/server$ npm i
~/TI0080_SGS/server$ npm install nodemon
```
2. Executar aplicação
``` bash 
~/TI0080_SGS/app$ nodemon app.js
~/TI0080_SGS/server$ nodemon server.js
```
